Android Debloat Script Generator

This tool will generate a script to debloat your phone based on the output of the PM command.

The resulting script will probably require root. I did this on a Sony Xperia XZ1 compact without unlocking the bootloader, so any temporary root will do the trick.

Prerequisites

  • A computer with ADB and relevant drivers installed
  • USB debugging enabled on the device

Tool

  1. Run adb shell pm list packages -f and paste the output into the text are below.
  2. Press the button.
  3. Select the packages you want to disable.
  4. Copy the output into a shell and run it.
